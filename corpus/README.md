# Corpus de conocimiento (sintético)

Base de conocimiento de **TeleNova**, una operadora de telecomunicaciones ficticia,
generada con IA para desarrollo, demo y evals de AgentPilot. Cualquier parecido con
tarifas u operadoras reales es coincidencia.

12 documentos Markdown que simulan la documentación típica que consulta un agente
de call center: catálogos, procedimientos, argumentarios y políticas.
